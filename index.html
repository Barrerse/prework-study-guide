HTML

  • The head element contains information about the webpage.
  • The body element represents the visible ontent shown to the user.
  • The h1 element should only be used once per page and it is the main title of the webpage.
  • The h2 element is used to highlight different parts of the webpage.
  • The li element is used to create bulletpoints on a list such as this one
  • The footer element contains information about the wesbite like the author, copyright, sitemap, etc.
  • To identify the webpage as a HTML document it is specified at the begining with the DOCTYPE element.
  • It is useful to view the live document and look for and solve bugs in real-time.
  • Identifying proper meta elements will give your webpage a good search engine rank.

CSS

  • CSS stands for Cascading Stule Sheets
  • CSS is used to style the wesbite
  • CSS is a file seperate from the HTML file, but it can be linked in HTML though the link element
  • CSS Rule is structured with a selector followed by delcaration blocks
  • The selector is the HTML element that will be styled (header) and the declaration blocks consits of the properties and values of the style (color:blue)

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git branch --delete branch-name: deletes branch
  • git add -A: adds modifcations in the current working branch. the -A falg indicates to add all changes.
  • git commit -m"add message here": creates a commit and the -m adds commit message to describe the change
  • git pull: command to receive a branch's modifcations to the local enviroment. For example "git pull origin main" would pull to the origin, which is the GitHub repo, in the main branch.
  • git push: pushes the changes we have locally to our remote GitHub Branch. For example "git push origin branch-name" would create a remote branch in GitHub

JavaScript

  • A variable is simply a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A function is simply a set of instructions that tells the computer how to perform a certain task.
  • An array is a single variable that is used to hold a group of data.
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
